# LibraryManagementSystem
1. DESCRIPTION
A terminal-based library management application built with Python and SQLite3. It allows managing books, authors, and loan transactions with a role-based system.

2. KEY FEATURES

Manager Access: Add/Delete books, Add authors, View all system records.

Member Access: List books, Borrow books, Return books.

Smart UI: Displays active loan IDs during the return process and author IDs during book entry.

Database: Uses relational tables (Books, Authors, Members, Loans).

3. SETUP AND RUN

First, run 'create_admin.py' to create an admin account.

Run 'main.py' to start the application.
